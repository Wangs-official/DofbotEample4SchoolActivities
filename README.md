## 项目简介

基于 Python + Tkinter 开发，功能参考下方列表

为了在学校活动展示机械臂写了这些代码，仅作为存档使用

## 注意事项

0. 此注意事项写给开发人员看的，也是只为了这个机械臂开发的，所以可能在别的平台上根本跑不了
1. 如果要使用颜色抓取功能，请使用正方体，剩下的这个机械臂抓不起来
2. 请关闭开机后自己开的四个窗口，要不然会用不了摄像头
3. 为了保证正常显示X11窗口，请在一切开始前运行`export DISPLAY=:0`。当然，如果你没有使用SSH，则无需打此命令
4. 本地远程开发时，请将电脑网口和机械臂网口直连，并在本机上配置IP为`10.0.0.1/24`，不要填写网关，如果你已经将电脑和机械臂连接到了路由器上即可无视
5. 请在一切开始前做好HSV校准，已经集成到UI里了，到现场也要校准一次
6. 大Vibe时代你猜这代码谁写的
7. 记得连声卡线，有条件加个独立麦克风

## 环境与说明

此代码仅适用于DOFBOT机械臂，很多代码都是从官方示例程序上扣下来并且进行二改的，所以一般不会出现环境问题，我也没有加什么奇奇怪怪的库

## 可用功能

1. 语音类（使用 你好小亚 唤醒）

    - 鼓掌
    - 点头
    - 跪下
    - 祈祷
    - 初始位置
    - 惊吓
    - 跳舞

2. 语音动作类（使用 你好小亚 唤醒）

    - 叠罗汉 （请参考下方图片放置方块）
        ![](https://www.yahboom.com/public/upload/upload-html/1766820113/image-20251124122319057.png)
    - 夹方块
    - 搬运（请参考下方图片放置方块）
        ![](https://www.yahboom.com/public/upload/upload-html/1766820113/image-20251124122756105.png)

3. 颜色追踪

    在主界面点击颜色追踪即可，摄像头会跟着方块一起走，HSV校准为独立校准，请点击“颜色学习”以学习颜色

4. 颜色抓取

    在主界面点击颜色抓取即可，将方块放在摄像头进行识别，并将大正方体放到对应的颜色位置，机械臂会自动抓取

## 开始运行

`python3 main.py`
